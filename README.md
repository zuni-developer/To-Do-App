# To-Do App (A mini practise project)

A simple and responsive To-Do application built with React to practice the core concepts of React development. This project focuses on understanding how state, props, components, and hooks work together to create an interactive application.

---

## Features

- Add new tasks
- Mark tasks as completed
- Delete tasks
- Add tasks by pressing **Enter**
- Shows an empty state when there are no tasks
- Browser tab updates with the number of remaining tasks using `useEffect`
- Clean and responsive UI

---

## React Concepts Practiced

This project helped me practice:

- Functional Components
- Props
- `useState`
- `useEffect`
- Event Handling
- Conditional Rendering
- List Rendering
- Component Reusability
- State Management

## Getting Started

Clone the repository

```bash
git clone https://github.com/zuni-developer/To-Do-App.git
```

Navigate into the project

```bash
cd To-Do-App
```

Install dependencies

```bash
npm install
```

Run the development server

```bash
npm run dev
```

Build for production

```bash
npm run build
```
